## Abstarct Class
- A class declared with abstract keyword
- abstract class cannot be instantiated(no object creation).
- It can contain
	-abstarct methods(no body)
	-normal methods(concrete methods)
	-static methods
	-final methods
	-variables,constructors etc..
- abstract methods 
	- method that is declared without implementation of a body
	- it has only declaration,ending with semicolon(;)
	- Mandatory overriding
	-cannot be private,static,final
- If class contains abstract methods then the class must be abstract.

##final keyword

- final variable(cannot be reassigned)
- final method(cannot be overriden)
- final class(cannot be inherited(extended))
