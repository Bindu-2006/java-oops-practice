## Access Modifiers
- access modifiers control who can access a class, method, variable or a constructor


### Types of Access Modifiers

- private(within the class)
- default(within the package)
- protected(within package + subclass different package)
- public(everywhere)
  
---

## Object Class

- Object is a super class of all classes in java.
- Every class automatically extends Object.

### Important Methods of Object Class

- toString()-returns String representation
- equals()-compares object
- getClass()-returns class name
- clone()-copies objects
- hashCode()-returns hash value
- finalize()-called before garbage collection
	-System.gc() is just a suggestion to clear data
